# مشروع البث المباشر (WebRTC + Socket.io)

هذا المشروع عبارة عن تطبيق بث مباشر متكامل يعتمد على تقنية **WebRTC** لنقل الفيديو والصوت في الوقت الفعلي، واستخدام **Socket.io** كخادم للإشارات (Signaling Server). التطبيق مصمم ليعمل بكفاءة على الهواتف المحمولة والمتصفحات الحديثة مع واجهة عربية بالكامل.

## المميزات الرئيسية

- **بث مباشر في الوقت الفعلي:** استخدام WebRTC لضمان أقل تأخير ممكن (Low Latency).
- **دعم المشاهدين المتعددين:** يمكن لمذيع واحد البث لعدة مشاهدين في آن واحد.
- **الحفاظ على نشاط الكاميرا (Mobile-Friendly):**
    - استخدام **Screen Wake Lock API** لمنع انطفاء الشاشة أثناء البث.
    - تقنية **Silent Audio Trick** لضمان استمرار عمل المتصفح في الخلفية ومنع Chrome من إيقاف الكاميرا.
- **واجهة مستخدم عصرية:** مصممة باستخدام Tailwind CSS وتدعم اللغة العربية (RTL).
- **جاهز للنشر:** يتضمن إعدادات النشر على منصة **Render**.

## هيكل المشروع

- `server.js`: خادم Node.js المسؤول عن الإشارات وتوجيه الاتصالات.
- `public/broadcast.html`: صفحة المذيع (بدء البث).
- `public/watch.html`: صفحة المشاهد (متابعة البث).
- `render.yaml`: ملف تكوين النشر التلقائي على Render.

## طريقة التشغيل محليًا

1. تأكد من تثبيت Node.js على جهازك.
2. قم بتثبيت التبعيات:
   ```bash
   npm install
   ```
3. ابدأ تشغيل الخادم:
   ```bash
   node server.js
   ```
4. افتح المتصفح على:
   - للمذيع: `http://localhost:3000/broadcast.html`
   - للمشاهد: `http://localhost:3000/watch.html`

## ملاحظات تقنية

- تم استخدام خوادم STUN المجانية من Google لضمان تجاوز جدران الحماية (Firewalls).
- المشروع يدعم منفذ التشغيل (PORT) من متغيرات البيئة لضمان التوافق مع منصات السحاب.
- تمت إضافة منطق إعادة الاتصال التلقائي في حال انقطاع اتصال الـ Socket.
